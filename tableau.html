Step 1: Choose & Download Your Tableau Version

First, you need the main Tableau application. Choose one of the two options below. You only need one!

Download Tableau Desktop

Who is this for?
Users who have a product key or want to start a 14-day free trial of the full software. It allows you to save your work privately on your computer.

Option 2: Tableau Public (Free Version)

This is the free version, perfect for learning and public sharing.

Go to Public Download Page

Who is this for?
Anyone who wants to learn Tableau for free! Important: You cannot save files locally. Your work is saved to your public online profile for anyone to see.

üßë‚Äçüè´ Step 3: Installation & Setup Guide

Follow along with our video guide for a smooth installation, or use the step-by-step instructions below. The process is the same for both Tableau Desktop and Public.

  1. Download Your Tools: Make sure you have downloaded both a Tableau installer (from Step 1) and a practice dataset (from Step 2).
  2. Install the Application: Find the downloaded installer file (it will start with 'Tableau...') in your 'Downloads' folder and double-click it. The installation wizard will appear. Simply follow the on-screen instructions (Accept the terms, click 'Install', etc.).
  3. Prepare Your Dataset:
    • If you downloaded the Day 1 Excel file, you're ready to go!
    • If you downloaded the Day 2 ZIP file, find it, right-click on it, and choose "Extract All...". This will create a new folder containing the Excel file you need.
  4. Launch Tableau: Open the Tableau application from your Start Menu or Desktop.
  5. Connect to Your Data:
    On the Tableau start screen, look for the "Connect" pane on the left.
    • Click on "Microsoft Excel".
    • A file browser will open. Navigate to where you saved your dataset and select it (either the Day 1 Excel file or the one you extracted from the Day 2 ZIP).
  6. Start Exploring: Congratulations! üéâ Your data is now loaded. You can start dragging and dropping data fields to create your first charts and dashboards.
