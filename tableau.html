Step-by-Step Guide:

  1. Download Tableau Desktop: Click the big green button in Step 1 above to download the installer.
  2. Install the Application: Find the downloaded file and double-click it. Follow the on-screen instructions (Accept terms, click Install, etc.).
  3. Download a Dataset: Choose either the Day 1 or Day 2 dataset from Step 2 and download it.
  4. (If you chose Day 2) Unzip the File: Find the downloaded `Day2_Dataset...zip` file. Right-click on it and choose "Extract All..." to create a new folder with the Excel file inside.
  5. Launch Tableau Desktop: Open the Tableau application from your Start Menu or Desktop.
  6. Connect to Your Data:
    • On the Tableau start screen, under "Connect," click on "Microsoft Excel."
    • Navigate to and select the dataset file you downloaded (either the Day 1 Excel file or the one you extracted from the Day 2 ZIP).
  7. Start Exploring: Congratulations! üéâ Your data is now loaded. You can start dragging and dropping fields to create charts and dashboards.
